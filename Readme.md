
# Quantity Measurement App — Frontend (HTML/CSS/JS) (feat/frontend-html)

Static frontend for the Quantity Measurement App built with HTML, CSS, and JavaScript.

## Pages

- **index.html** — Landing page
- **login.html** — User login (Google OAuth & manual)
- **signup.html** — User registration
- **dashboard.html** — Measurement panel (convert, add, subtract, compare, divide)

## Features

- JWT-based authentication
- Google OAuth login support
- Unit conversion (Length, Weight, Volume, Temperature)
- Operations: Convert, Add, Subtract, Compare, Divide
- Dark / Light mode toggle
- Responsive design

## Tech Stack

- HTML5
- CSS3 (Tailwind CSS)
- Vanilla JavaScript

## Backend

This frontend connects to the Spring Boot REST API running at `http://localhost:8080`.
